In the summer of 2026, I was preparing an application for an interdisciplinary art and science residency. The application required something I had never assembled before: a portfolio—not of publications, but of ideas.

For years, my work had been scattered across scientific papers, conference presentations, field notebooks, software prototypes, photographs, and unfinished essays. They all described different parts of the same journey, yet none of them revealed the whole.

The residency application became a catalyst.

Not because I wanted another website, but because I realized I needed a place where landscapes, science, computation, writing, and personal reflection could finally exist in conversation with one another.

What began as a portfolio gradually evolved into something much larger.

Thinking Across Scales.

This website was not planned over years.

It was built over two weeks.

What once required months of web development can now be built through modern open-source tools, AI-assisted development, and reproducible publishing workflows.

The technology accelerated the construction.

The ideas had been developing for decades.

Built with:

- Quarto
- GitHub
- Mapbox
- Markdown
- Git
- AI-assisted development
- Cloud-based workflow

The website itself became another field experiment—an exploration of how modern computational tools can expand the practice of scientific communication.
