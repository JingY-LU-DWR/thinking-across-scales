# World Map Component

## Purpose

Create a reusable SVG world map component for a Quarto website.

The map should become the primary navigation interface for the Journeys section.

It should be lightweight.

No JavaScript.

Use inline SVG.

Design Style

Minimal

Editorial

Muted earth tones

Thin coastlines

Lots of whitespace

Do not use Google Maps.

Do not use Leaflet.

Do not use Mapbox.

Locations

Everglades

Yellowstone

Yunnan

Tibet

Maine

Sacramento

Each location should appear as a circular marker.

Each marker should link to its corresponding Journey page.

The SVG should scale responsively.
